# 화면 정의서 — RF4 선호 어종 대시보드

문서 버전: v1.0 (2026-06-12)
근거 문서: PRD v1.1
범위: P0 화면 3개 (온보딩 / 메인 대시보드 / 어종 상세)

---

## 0. 공통

### URL 구조
```
/                     메인 대시보드 (선호 어종 0개면 온보딩으로 리다이렉트)
/onboarding           어종 선택 (온보딩 겸 어종 관리)
/species/{어종명}      어종 상세
```
- 시간창은 쿼리 파라미터: `?window=today|3d` (기본 3d). 화면 간 이동 시 유지

### 공통 요소
- 헤더: 사이트명(좌) / 마지막 수집 시각(좌, 작게) / 시간창 토글 **오늘·3일**(우) / 어종 관리 버튼(우)
- 마지막 수집이 24시간 이상 지났으면 수집 시각을 경고색으로 표시 ("데이터가 오래됨")

### 등급 뱃지 (전 화면 공통)
| 등급 | 표기 |
|------|------|
| 레어트로피 | 파랑 뱃지 "레어" |
| 트로피 | 녹색 뱃지 "트로피" |
| 일반 | 뱃지 없음 |

### 활성 상태 뱃지
| 상태 | 색 | 카드 표현 |
|------|-----|----------|
| 강한 활성 | 녹색 | 전체 정보 표시 |
| 활성 | 파랑 | 전체 정보 표시 |
| 불명 | 주황 | 대표 미끼 노출하되 낮은 일관성 %를 함께 표기 (확정) |
| 비활성 | 회색 | 한 줄 축소 + 흐림 처리, "표본 부족 (n건)" 표기 |

## 1. 온보딩 / 어종 관리 (`/onboarding`)

**목적**: 선호 어종 등록. 첫 방문(선호 0개)과 이후 관리 양쪽에서 같은 화면 사용.

### 구성
1. 안내 문구 (첫 방문일 때만): "노리는 어종을 등록하면 대시보드가 만들어집니다"
2. 검색창: 어종명 부분 일치 검색 (한글 자모 단위 즉시 필터)
3. 어종 목록: 수집된 어종 전체(약 251종), 가나다순. 행 = 어종명 + 별표 토글
4. 하단 고정 바: "선택한 어종 N종 · 대시보드 보기" 버튼

### 상태
- 별표 클릭 → 즉시 저장 (저장 버튼 없음)
- 선호 0종이면 "대시보드 보기" 비활성화
- 검색 결과 0건: "해당 어종 없음" + 검색어 지우기

## 2. 메인 대시보드 (`/`)

**목적**: 접속 즉시 "오늘 뭘 잡으러 갈지" 결정. 선호 어종이 활성도 순으로 정렬된 카드 목록.

### 구성 (와이어프레임 확정안)
1. 공통 헤더
2. 어종 카드 목록 — 활성도 점수 내림차순, 비활성은 항상 최하단
   - 카드: 어종명 / 활성 상태 뱃지 / 활성도 점수(우측)
   - 근거 줄: 트로피·일반 건수 / 미끼 일관성 % / 대표 장소 / 대표 미끼
   - 카드 클릭 → 어종 상세로 이동
3. 비활성 카드: 한 줄 축소형 (어종명 + "표본 부족 n건")

### 상태
| 상황 | 처리 |
|------|------|
| 선호 어종 0개 | `/onboarding`으로 리다이렉트 |
| 시간창 내 전 어종 비활성 | 카드는 그대로 표시 + 상단 안내 "시간창을 3일로 넓혀보세요" (오늘 창일 때) |
| DB에 기록 자체가 없음 | "수집된 데이터 없음 — collector를 먼저 실행하세요" 안내 |

### 인터랙션
- 시간창 토글 → 같은 화면 리로드 (`?window=` 변경), 점수·상태 재계산
- 어종 관리 버튼 → `/onboarding`

## 3. 어종 상세 (`/species/{어종명}`)

**목적**: 추천받은 어종의 출조 준비 — 무슨 미끼, 어느 장소, 목표 무게.

### 구성 (와이어프레임 확정안)
1. 헤더 줄: 뒤로가기 / 어종명 / 활성 상태 뱃지 / **트로피 기준선 표기** (예: "트로피 28kg · 레어 40kg") / 우측 "트로피만" 토글
2. 미끼 순위 블록: 시간창 내 미끼별 기록 수 순위 (점유율 막대 + 건수 + %). 색상코드까지 원본 그대로 표기
3. 장소 분포 블록: 수역별 기록 수
4. 최근 트로피 기록 블록: 트로피이상 기록 최신순 (무게 + 등급 뱃지 + 날짜 + 장소)

### 상태
| 상황 | 처리 |
|------|------|
| "트로피만" 토글 ON | 미끼 순위·장소 분포의 집계 모집단을 트로피이상으로 한정 |
| 시간창 내 기록 0건 | "최근 N일 기록 없음" + 시간창 전환 제안 |
| 트로피 기준 미등록 어종 | 기준선 미표기, 등급 뱃지 생략 (전부 일반 취급) |
| 선호 미등록 어종 직접 URL 접근 | 정상 표시 + "선호에 추가" 별표 노출 |
